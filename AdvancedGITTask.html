Task 1: Parallel Feature Branch Integration (Ana & Luis)

Ana: Merge ft_one into main, with PR feedback

  • Create commits #1 and #4 in feature branch ft_one
  • git checkout main Switch to main to fetch updates
  • git pull origin main Get the latest main branch changes
  • git checkout ft_one Return to ft_one for rebasing
  • git rebase main ft_one Rebase ft_one onto main
  • git add <conflict_file> & git rebase --continue Resolve each conflict as prompted
  • git push origin ft_one Push completed rebased branch to remote
  • Create a pull request: base=main, compare=ft_one, add title/description
  • Reviewer reviews PR and requests changes
  • Perform commit #7 to resolve review issues
  • git push origin ft_one Update PR with fixes
  • Reviewer approves the updated PR
  • ft_one successfully merged into main via PR
