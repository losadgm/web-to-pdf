Registro de cambios: circuito.xsd

Configuración de namespace y codificación

23 de octubre de 2025
  • Se añadió el atributo xmlns="http://www.uniovi.es" al elemento xs:schema
  • Se añadió el atributo targetNamespace="http://www.uniovi.es" para referenciar al namespace del documento que se validará con este XML Schema
  • Se modificó el atributo de codificación de encoding="windows-1252" a encoding="utf-8" en la instrucción de procesamiento XML

Cambio de tipos de elementos

24 de octubre de 2025
  • El elemento largo ahora es de tipo xs:decimal
  • El elemento ancho ahora es de tipo xs:decimal
  • Las coordenadas latitud y longitud ahora son de tipo geoCoordenada
  • El elemento altitud ahora es de tipo xs:decimal
  • El elemento sector ahora es de tipo xs:integer

Actualización de atributos

24 de octubre de 2025
  • El atributo fecha del elemento carrera ahora es de tipo xs:date
  • El atributo horaInicio del elemento carrera ahora es de tipo xs:time
  • El atributo numeroVueltas del elemento carrera ahora es de tipo xs:decimal
  • El atributo tiempoEmpleado del elemento vencedor ahora es de tipo xs:duration
  • El atributo principal del elemento patrocinador ahora es de tipo xs:boolean y su valor por defecto es false

Modificación de ocurrencias

24 de octubre de 2025
  • El atributo maxOccurs del elemento fotografías para la secuencia de fotografía se ha establecido en 5
  • El atributo maxOccurs del elemento videos para la secuencia de video se ha establecido en 3
  • El atributo minOccurs del elemento referencias para la secuencia de referencia se ha establecido en 3
