Guía de Configuración de Volumio en Raspberry Pi 3

Volumio es un software de código abierto diseñado para convertir dispositivos como la Raspberry Pi en un reproductor de música en red de alta calidad, que puedes controlar desde tu teléfono o navegador web.

1. Instalación Inicial

Antes de configurar, necesitas instalar la imagen de Volumio en una tarjeta microSD.

  1. Descarga Volumio: Ve a la página oficial de Volumio y descarga la imagen gratuita correspondiente a la Raspberry Pi.
  2. Flashea la tarjeta SD: Utiliza una herramienta como Raspberry Pi Imager o BalenaEtcher para grabar la imagen descargada en una tarjeta microSD. Se recomienda una tarjeta de al menos 8GB.
  3. Inserta la tarjeta y arranca: Coloca la microSD en tu Raspberry Pi 3 y enciéndela. El primer arranque puede tardar unos minutos mientras el sistema se autoconfigura.

2. Configuración de Red

Una vez que la Raspberry Pi ha arrancado, necesitas conectarla a tu red.

Opción A: Conexión por Wi-Fi (Método recomendado)

Al iniciarse por primera vez sin una conexión de red por cable, Volumio crea su propio punto de acceso Wi-Fi.

  • Desde tu ordenador o teléfono, busca las redes Wi-Fi disponibles y conéctate a una llamada Volumio.
  • La contraseña para esta red es volumio2.
  • Una vez conectado, abre un navegador web y accede a la dirección http://volumio.local.
  • Se abrirá la interfaz de Volumio. Ve al menú de Configuración (el icono del engranaje) y selecciona Red.
  • Busca tu red Wi-Fi doméstica, selecciónala, introduce tu contraseña y haz clic en conectar. Volumio se reiniciará y se conectará a tu red.

Opción B: Conexión por Ethernet

Si prefieres una conexión por cable, simplemente conecta un cable Ethernet a tu Raspberry Pi. Después de unos momentos, podrás acceder a la interfaz web desde otro dispositivo en la misma red visitando http://volumio.local.

3. Asistente de Configuración

Al acceder a la interfaz web por primera vez, un asistente te guiará a través de la configuración inicial:

  • Idioma: Selecciona tu idioma preferido.
  • Nombre del dispositivo: Dale un nombre a tu reproductor Volumio (por ejemplo, "Sala de Estar").
  • Salida de audio: Elige dónde se reproducirá el sonido. Para la mayoría, será "Headphones" (para la salida de jack de 3.5 mm) o HDMI si está conectado a un receptor A/V. Si tienes una tarjeta de sonido DAC, la podrás seleccionar aquí más tarde.

4. Documento de Configuración (config.txt)

La mayoría de los ajustes de Volumio se gestionan a través de su interfaz web. No se debe usar la herramienta raspi-config tradicional, ya que está diseñada para Raspberry Pi OS, no para Volumio.

Sin embargo, para ajustes de hardware de bajo nivel (como activar soporte para un DAC específico o ajustar la memoria de la GPU), puedes editar el fichero /boot/config.txt. Este fichero se encuentra en la partición de arranque de la tarjeta SD, por lo que puedes editarlo desde un ordenador.

A continuación se muestra un ejemplo de un fichero config.txt para una Raspberry Pi 3 con Volumio:

# Fichero de configuración base para Volumio en Raspberry Pi 3

# --- Configuración Esencial ---
initramfs volumio.initrd
# Asigna memoria mínima a la GPU, ya que Volumio no la necesita
gpu_mem=16 
# Permite mayor consumo de corriente en los puertos USB para discos duros externos
max_usb_current=1 
# Desactiva la pantalla de arranque de Raspberry Pi
disable_splash=1 

# --- Configuración de Audio ---
# Activa el hardware de audio por defecto y la comunicación I2C para DACs
dtparam=audio=on
dtparam=i2c_arm=on

# --- Ejemplo para un DAC (Tarjeta de Sonido) ---
# Si tienes un DAC como el HiFiBerry DAC+, descomenta la siguiente línea
# dtoverlay=hifiberry-dacplus

# --- Ejemplo para un receptor IR (mando a distancia) ---
# Habilita un receptor de infrarrojos en el pin GPIO 26
# dtoverlay=lirc-rpi:gpio_in_pin=26

Este es un fichero de configuración estable y básico. Solo necesitas añadir o modificar las líneas dtoverlay si conectas hardware adicional como una tarjeta de sonido (DAC) o una pantalla específica.

5. Añadir tu Música

Una vez que Volumio está en red y configurado, puedes empezar a añadir tu música:

  • USB: Conecta un disco duro externo o una memoria USB con tus archivos de música a uno de los puertos USB de la Raspberry Pi. Volumio lo detectará y añadirá automáticamente la música a la biblioteca.
  • NAS (Almacenamiento en Red): Si tienes tu música en un servidor NAS, puedes añadirlo desde la sección "Fuentes" en el menú de configuración.
  • Plugins: Volumio también soporta plugins para servicios de streaming como Spotify, lo que te permite integrar tu cuenta.
